---
title: "macOS"
description: "Install Kisuke Connect on macOS"
---

The current version of Kisuke Connect requires macOS 12 (Monterey) or later.

1. Download Kisuke Connect using one of the following methods:
   - [Download from the Kisuke website](https://beta.kisuke.dev/downloads/)
   - Install via Homebrew:

     ```bash
     brew install kisuke-connect
     ```
2. Open Kisuke Connect from Applications and follow the onscreen instructions.
3. Sign in with the same account you use in Kisuke.

## Troubleshooting

If Kisuke Connect does not appear in Applications after installing via Homebrew, run:

```bash
brew link kisuke-connect
```
