## Install options

### Desktop app

Download and install the Kisuke Connect desktop app from the [Kisuke downloads page](https://beta.kisuke.dev/downloads/). Available for macOS, Windows, and Linux. The desktop app runs in the background and appears in your menu bar (macOS) or system tray (Windows/Linux).

### Package managers

- **macOS** — Install via [Homebrew](https://brew.sh):
  ```bash
  brew install kisuke-connect
  ```
- **Linux** — Install via your distribution's package manager (apt, dnf, yay) or use the install script:
  ```bash
  curl -fsSL https://kisuke.app/install.sh | sh
  ```

### CLI (headless)

For servers and automated environments without a desktop, install the CLI via npm:

```bash
npm install -g @kisuke/cli@dev
```

Run `kisuke` to authenticate and start the daemon. See [Install CLI](/installation/cli) for details.

## Uninstall

To uninstall Kisuke Connect, remove it using the same method you used to install it (e.g., drag to Trash on macOS, uninstall from Settings on Windows, or `apt remove kisuke-connect` on Linux).
