---

The CLI is the headless version of Kisuke Connect — the same daemon, without the desktop GUI. Use it for servers and automated environments.

## Install

```bash
npm install -g @kisuke/cli@dev
```

## Run

Once installed, run `kisuke` in your terminal:

```bash
kisuke
```

The CLI will automatically open a browser window to authenticate your account. Sign in and the terminal will confirm:

```
✓ Logged in as you@example.com
```

You'll then be prompted to install and start the Kisuke Connect daemon:

```
Do you wanna install and start Kisuke?
By selecting `Yes, I agree`, you agree to:
  Privacy: https://beta.kisuke.dev/privacy
  Terms: https://beta.kisuke.dev/terms

> Yes, I agree
  Quit
```

Select **Yes, I agree** and the daemon will start running in the background.

<ParamField body="KISUKE_STATE_DIR" type="string" default="/var/lib/kisuke">
