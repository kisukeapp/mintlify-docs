New users can follow the [Quickstart](/quickstart) guide to set up Kisuke and Kisuke Connect in minutes.

## Install instructions

The following topics provide platform-specific installation instructions for the Kisuke app.

- [Install on iOS](/installation/ios)
- [Install on Android](/installation/android)
- [Install on macOS](/installation/app-macos)
- [Install on Windows](/installation/app-windows)
- [Install on Linux](/installation/app-linux)
- [Use the Web App](https://web.kisuke.app) — no install needed, open Kisuke directly in your browser

## Install options

### App stores

- **iOS** — Download from the [App Store](https://apps.apple.com/app/kisuke). Requires iOS 15.0 or later.
- **Android** — Download from [Google Play](https://play.google.com/store/apps/details?id=app.kisuke). Requires Android 8.0 (Oreo) or later.

### Direct download

- **macOS, Windows, Linux** — Download the latest version from the [Kisuke downloads page](https://beta.kisuke.dev/downloads/).

### Web app

No install required. Open [web.kisuke.app](https://web.kisuke.app) in any modern browser.

## Next steps

Once you've installed Kisuke, install [Kisuke Connect](/installation/kisuke-connect) on the computer you want to access remotely.
