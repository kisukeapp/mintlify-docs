## Install instructions

The following topics provide platform-specific installation instructions for the Kisuke app.

- [Install on iOS](/installation/ios)
- [Install on Android](/installation/android)
- [Install on macOS](/installation/app-macos)
- [Install on Windows](/installation/app-windows)
- [Install on Linux](/installation/app-linux)

## Next steps

Once you've installed Kisuke, install [Kisuke Connect](/installation/kisuke-connect) on the computer you want to access remotely.
