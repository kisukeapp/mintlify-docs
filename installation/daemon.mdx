Kisuke Connect is a lightweight desktop app that runs in the background and securely bridges your computer to the Kisuke mobile app over an end-to-end encrypted connection. It gives Kisuke access to your terminal, files, and editor.

## Download

<Card title="Download Kisuke Connect" icon="download" href="https://beta.kisuke.dev/downloads/">
  Get the latest version for macOS, Windows, or Linux.
</Card>

## Setup

<Tabs>
  <Tab title="macOS">
    Requires macOS 12 (Monterey) or later.

    1. [Download Kisuke Connect](https://beta.kisuke.dev/downloads/) and open the `.dmg` installer.
    2. Drag **Kisuke Connect** into your Applications folder.
    3. Open Kisuke Connect from Applications — it will appear in the menu bar.
    4. Sign in with the same account you use in Kisuke.

    **Troubleshooting:** If macOS blocks the app, go to **System Settings** > **Privacy & Security** and click **Open Anyway**.
  </Tab>
  <Tab title="Windows">
    Requires Windows 10 (1903) or later.

    1. [Download Kisuke Connect](https://beta.kisuke.dev/downloads/) and run the installer.
    2. After installation, Kisuke Connect will appear in the system tray.
    3. Sign in with the same account you use in Kisuke.

    **Firewall:** Kisuke Connect may prompt you to allow network access. Click **Allow** when the Windows Firewall dialog appears. If you need to configure it manually: open **Windows Defender Firewall** > **Allow an app through firewall** > find **Kisuke Connect** and enable both Private and Public networks.

    **Troubleshooting:** If Kisuke Connect does not appear in the system tray, check the overflow area by clicking the **^** arrow in the taskbar.
  </Tab>
  <Tab title="Linux">
    Requires Linux kernel 4.19 or later.

    1. [Download Kisuke Connect](https://beta.kisuke.dev/downloads/) and install the package for your distribution (`.deb` or `.rpm`).
    2. Launch Kisuke Connect from your application menu — it will appear in the system tray.
    3. Sign in with the same account you use in Kisuke.

    **Troubleshooting:** If the tray icon does not appear, make sure your desktop environment supports system tray icons (e.g., install `gnome-shell-extension-appindicator` on GNOME).
  </Tab>
</Tabs>
