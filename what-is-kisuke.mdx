Your best ideas don't wait — so Kisuke brings a full development environment to wherever you are, whether that's your phone, your laptop, or your desktop. Write code, run commands, and chat with AI assistants on iOS, Android, macOS, Windows, Linux, and the web.

## Kisuke vs Kisuke Connect

Kisuke has two parts that work together:

|                  | Kisuke                                                                                                        | Kisuke Connect                                  |
| ---------------- | ------------------------------------------------------------------------------------------------------------- | ----------------------------------------------- |
| **What it is**   | The app you use                                                                                               | Desktop / server daemon                         |
| **Runs on**      | iOS, Android, macOS, Windows, Linux, Web                                                                      | macOS, Windows, Linux, servers                  |
| **Role**         | The client you use                                                                                            | The server you connect to                       |
| **Install from** | TestFlight (iOS), Google Play, [downloads](https://beta.kisuke.dev/downloads/), [web](https://web.kisuke.app) | [downloads](https://beta.kisuke.dev/downloads/) |

**Kisuke** is the app you use on any device — phone, tablet, desktop, or browser. **Kisuke Connect** is a lightweight background daemon that runs on your desktop or server and exposes your terminal, files, and editor to the app — over an end-to-end encrypted connection across 14+ global regions.

You sign in to both with the same account and they find each other automatically.

## Key Features

### AI Chat

Kisuke includes a built-in AI chat interface that supports multiple providers including Anthropic, OpenAI, Google, xAI, and more.

Sign in with your existing account to start coding with AI assistance from any device.

### Terminal & Shell

Access a full terminal environment from any device. Run commands, execute scripts, and manage your development workflow — locally or connected remotely via Kisuke Connect.

### Code Editor

A code editor with:

- Syntax highlighting
- File management
- Project navigation

### Kisuke Connect

Connect to your desktop or server using Kisuke Connect. This allows you to:

- Access your development machine from anywhere
- Run commands on remote servers
- Edit files directly on remote machines

## How It Works

1. **Install Kisuke** on [iOS](https://apps.apple.com/app/testflight/id899247664) (via TestFlight), [Android](https://beta2.kisuke.dev/downloads/), [Mac](https://beta2.kisuke.dev/downloads/), [Windows](https://beta2.kisuke.dev/downloads/), [Linux](https://beta2.kisuke.dev/downloads/), or use the [web app](https://web.kisuke.app)
2. **Sign in** with Google or GitHub
3. **Install Kisuke Connect** on your [Mac](https://beta2.kisuke.dev/downloads/), [Windows](https://beta2.kisuke.dev/downloads/), [Linux](https://beta2.kisuke.dev/downloads/) machine or server — sign in with the **same** account
4. **Start coding** with the [terminal](/features/terminal), [editor](/features/editor), and [browser](/features/browser)

To use [**AI chat**](/features/coding-with-ai), sign in separately with your AI provider account.

- **Server management** - Connect to servers from any device via Kisuke Connect
- **Quick fixes** - Make hotfixes from your phone, tablet, desktop or browser
- **Learning** - Practice coding with AI assistance anywhere
